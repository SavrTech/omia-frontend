Key Pricing Factors

1. Number of Projects

Base pricing on project volume:

  • • 5-10 projects: $299-499/mo base
  • • 11-25 projects: $500-899/mo
  • • 26-50 projects: $900-1,499/mo
  • • 51-100 projects: $1,500-2,499/mo
  • • 100+ projects: Custom pricing (typically $25-35 per project/mo)

2. Team Size

Additional cost per user over base limits:

  • • Up to 15 users: Included in base
  • • 16-50 users: Add $15-25 per additional user/mo
  • • 51-100 users: Add $10-15 per additional user/mo
  • • 100+ users: Negotiated volume discount

3. Support Level

Support tier add-ons:

  • • Standard support (email, 24hr response): Included
  • • Priority support (phone + email, 4hr response): +$200-400/mo
  • • Premium support (24/7, 1hr response): +$500-800/mo
  • • Dedicated success manager: +$1,000-2,000/mo

4. Advanced Features

Premium feature pricing:

  • • SSO/SAML integration: +$100-200/mo
  • • White-label deployment: +$300-500/mo
  • • API access: +$200-400/mo
  • • On-premise deployment: +$2,000-5,000/mo (plus setup fee)
  • • Custom integrations: Quote per integration ($5K-25K one-time)
  • • Compliance certifications (SOC2, HIPAA): +$500-1,500/mo

5. Contract Length

Discount structure for commitment:

  • • Month-to-month: Full price
  • • 6-month commitment: 5% discount
  • • 12-month commitment: 10-15% discount
  • • 24-month commitment: 15-20% discount
  • • 36-month commitment: 20-25% discount

Sales Negotiation Guidelines

Discount Authority

  • • Sales reps: Up to 10% discount without approval
  • • Sales managers: Up to 20% discount
  • • Director approval required: 20-30% discount
  • • C-suite approval required: 30%+ discount
